백엔드를 위한 DJANGO REST FRAMEWORK with 파이썬 / 영진닷컴 / 권태형 저

1.  python -m venv myvenv

2.  myvenv/Scripts/activate

3.  pip install django djangorestframework django-cors-headers

4.  django-admin startproject myboard .

5.  python manage.py createsuperuser
    (james / ~)

6.  python manage.py startapp users

7.  pip install pillow, axios, django-filter

8.  npm create vite@7
